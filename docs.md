Permission to run npm in terminal of VS Code
cmd: Set-ExecutionPolicy Bypass -Scope Process -Force

Create a Vue.js project:
cmd: npm create vite@latest

Install Tailwind CSS
https://tailwindcss.com/docs/installation/using-vite
Their document is important. It is related in files: 
- vite.config.js
- src / style.css
- App.vue
cmd: cd project-name
cmd: npm install tailwindcss @tailwindcss/vite

Run the project
cmd: npm run dev

Install Tailwind CSS Icons
cmd: npm i @heroicons/vue

Install Tailwind CSS Animation Library
cmd: npm i aos

In search bar of VS Code 
Copy URL: http://localhost:5173/
